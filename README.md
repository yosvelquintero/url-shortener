# README

## url-shortener

URL-shortener service to shorten URLs

Built on top of [NestJS](https://nestjs.com/), a progressive [Node.js](https://nodejs.org) framework for building efficient and scalable server-side applications.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
